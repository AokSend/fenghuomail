## 三、主要功能与特色

### 1. 免费域名邮箱  
用户在注册后即可获赠一个永久免费的企业邮箱额度，方便个人或团队体验和试用。  

### 2. 按月购买，随用随付  
打破传统邮箱必须按年购买的限制，烽火邮箱支持 **月度订阅**，用户可以随时开通、随时停用，极大地降低了使用门槛和成本。  

### 3. 一个起买，拒绝捆绑  
很多邮箱服务商要求至少购买 5 个邮箱账号，而烽火邮箱允许 **一个账号起买**，真正做到按需使用，灵活方便。  

### 4. 别名邮箱归一  
可以为一个邮箱账号配置多个别名，所有发往别名邮箱的邮件都会自动汇总到主邮箱中。  
例如：  
- `sales@company.com`  
- `market@company.com`  
- `info@company.com`  
均可归集到同一个主邮箱，既简化了管理，又提高了收件效率。  

### 5. 群组邮箱分流  
群组邮箱支持一对多的邮件分发。发件人只需发送到群组地址，群组成员即可同时收到邮件，非常适合团队协作。  

### 6. 无限域名邮箱  
烽火邮箱支持 **无限邮箱功能**，即绑定一个主邮箱后，任意前缀的邮箱都能收信，所有邮件都会集中到主邮箱中。  
例如：  
- `user1@company.com`  
- `user2@company.com`  
- `support@company.com`  
- `random123@company.com`  
全部会进入同一个主邮箱，极大地提升了灵活性。  

### 7. 安全与稳定  
烽火邮箱自带反垃圾机制，配合高权重的纯净 IP 池，有效拦截垃圾邮件，保证邮件传输的安全性和送达率。
